# 🎨 Slick Themes

A collection of vibrant, dark color themes for Visual Studio Code. Each theme features a distinct accent color applied consistently across the entire UI — editor, sidebar, activity bar, tabs, terminal, and more — for a fully immersive coding experience.

## Installation

### From VSIX

1. Download the `.vsix` file from the [releases](https://github.com/null-terminate/slick-themes/releases) page
2. In VS Code, open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
3. Run **Extensions: Install from VSIX...**
4. Select the downloaded `.vsix` file

### From Source

1. Clone this repository
2. Copy the folder to your VS Code extensions directory
3. Restart VS Code

## Activating a Theme

1. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Search for **Preferences: Color Theme**
3. Pick any **Slick** theme from the list

## Features

- **11 color variants** — pick the vibe that suits your mood
- **Full UI theming** — title bar, activity bar, sidebar, tabs, status bar, terminal, menus, breadcrumbs, notifications, and more are all styled to match
- **Dark backgrounds** — easy on the eyes for long coding sessions
- **Consistent syntax highlighting** — readable token colors across all themes
